# CutieCats Search

A responsive React app that generates cat cards with the names and emails fetched from the API. Users can search the cats by name using the search bar.

## Link
### https://cutiecats-search.vercel.app/

## Tech Stack

- React
- TypeScript
- JavaScript
- HTML
- CSS
